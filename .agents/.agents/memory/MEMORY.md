# 🧠 Memória Global e Preferências do Projeto

## 🇵🇹 Regra de Ouro do Idioma (MANDATÓRIO)
- A comunicação com o utilizador tem de ser FEITA EXCLUSIVAMENTE em Português de Portugal (PT-PT).
- É estritamente proibido utilizar termos, sintaxe ou construções gramaticais do Português do Brasil (PT-BR).
- **Mapeamento de vocabulário obrigatório:**
  - `tela` ❌ → `ecrã` ✅
  - `arquivo` ❌ → `ficheiro` ✅
  - `mouse` ❌ → `rato` ✅
  - `usuário` ❌ → `utilizador` ✅
  - `time` ❌ → `equipa` ✅
  - `parabéns` / `legal` ❌ → `excelente` / `muito bem` ✅
- **Proibição de Gerúndios:** Nunca utilizar gerúndios estilo PT-BR (ex: "estou fazendo", "estou criando"). Utilizar sempre o infinitivo com a preposição "a" (ex: "estou a fazer", "estou a criar").
- *Nota Técnica:* Todo o código, comentários no código e mensagens de commit do Git devem manter-se rigorosamente em Inglês.

## 📌 Protocolo de Registo de Memória (/remember)
> **Instrução para os Agentes:** Sempre que o utilizador invocar o comando `/remember [informação]` ou pedir para memorizar um detalhe arquitetónico ou preferência, deves OBRIGATORIAMENTE utilizar a ferramenta `Edit` ou `Write` para adicionar essa linha com uma lista de pontos (`-`) diretamente por baixo da secção **"Registos Guardados"** deste ficheiro. Nunca confirmes a memorização no chat sem antes teres editado fisicamente este ficheiro no disco!

## 💾 Registos Guardados
- O utilizador prefere soluções diretas, automatizadas e sem necessidade de passos manuais repetitivos.
- (Os agentes devem adicionar novas memórias por baixo desta linha)
